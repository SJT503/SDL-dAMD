# SDL-dAMD

Code repository for: *A measurable trustworthiness audit for AI-driven scientific discovery* (Sheng et al., under review at npj Digital Medicine, 2026).

The complete code (audit harness, Bradley–Terry–Luce implementation, PubMed round-trip scorer, position-bias detection rule, figure-generation scripts) will be released here upon acceptance. The audit-trail data deposit is available at Zenodo: [10.5281/zenodo.20674265](https://doi.org/10.5281/zenodo.20674265).

For peer review, the complete source code is available to the editors and reviewers on request. Contact: jtsheng@stu.edu.cn
